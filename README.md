# Moldeojs Interface

Interface Component for [MoldeoJS](http://github.com/moldeo/moldeojs).

## Installation
`npm install`

## Serve
`npm start`

---
## moObjects
* *moDefault*: Default object for testing graphics, with not functions
* *moErase*: Simple erasing background object
* *moIcon*: Simple graphic object
